Frequently Asked Questions (FAQ)

  • 1. How do I add URLs to API Monitor?
    To add URLs, simply enter the desired URLs in the input field within the app, one per line, and click the "Apply" button to start monitoring.
  • 2. What do the colors on the status squares mean?
    The colors represent the status of each URL: Green indicates a successful response, while Red indicates a failure or timeout.
  • 3. How often does API Monitor check the status of URLs?
    By default, API Monitor checks the status of URLs every 15 seconds, but this interval may be customizable in future versions.
  • 4. What is the request timeout for API checks?
    API Monitor has a request timeout of 5 seconds. If a URL does not respond within this time frame, it is marked as a failure, and the status square will turn red.
  • 5. Can I export or save my list of monitored URLs?
    Currently, API Monitor stores the list of URLs locally on your device, but it does not yet support exporting the list.
